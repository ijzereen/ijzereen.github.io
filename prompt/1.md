# Tweaks

내가 Tweaks를 통하여 여러 시안을 조절할 수 있도록 Tweak을 만들어줘.

1. Pixel / 깔끔한 디자인
현재는 전체적으로 Pixel 이미지를 바탕으로 디자인 되어 있는데, 현대의 macos같이 깔끔한 디자인을 채용할 때와 Tweak를 이용하여 비교하고 싶어.

2. 한글 / 영어
사용자의 기반 언어에 따라 한글 / 영어를 설정할 수 있도록 하고싶은데, 우리가 맥의 위의 상태창에서 한/영 전환하는 것처럼 그렇게 하게 하고 싶거든. 그래서 한글도 지원할 수 있도록 해줘.

3. 전체적인 테마 색깔
현재 초록색 바탕화면 기준으로 테마가 컬러 테마가 정해져 있는데, 여러 컬러를 Tweak에서 시험해보고 싶어. 너가 어느정도의 대표적인 디자인 조합들을 설정해놓은 뒤에 내가 테스트해볼 수 있게 해줘.

4. 모바일 접속
모바일 환경에서 접속하면 window8의 모바일 폰 디자인처럼 타일 위주의 디자인이 되었으면 좋겠어. 이 것을 반영해줄 수 있을까?
