# 개요

포트폴리오 겸 블로그 용도로 사용할 github 블로그

# 디자인

바이오하자드 7의 디자인을 차용하고 싶어.


블로그를 들어오면 클라이언트는 어떤 책상 앞에 앉아있고, 책상에는 낡은 맥북과 함께 서류 철이 하나 있어. 이때 바이오하자드처럼 1인칭 3D 뷰여야 하고, 사용자의 커서에 따라 손전등이 비추는 구조였으면 좋겠어.

그리고 특정 오브젝트를 클릭하면 바이오하자드에서 오브젝트를 클릭한 것 처럼 오브젝트가 중앙으로 와 팝업처럼 뜨고, 그걸 한번 더 클릭하면 해당 메뉴로 이동할 수 있도록 해줘.

서류 철을 클릭하면 내 포트폴리오 글로 이동하고,
컴퓨터를 클릭하면 내 기술블로그로 이동할 수 있도록 해줘
